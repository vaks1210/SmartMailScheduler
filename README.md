# SmartMailScheduler – AI Calendar Assistant

SmartMailScheduler is a smart Chrome extension that enhances productivity by automatically detecting and scheduling events from your email inbox into your Google Calendar.

No more copying dates or switching tabs — with one click, important appointments are extracted and added using AI parsing and natural language understanding.

### Features

-  Parses emails to detect dates, times, and event descriptions.
-  Integrates directly with Google Calendar API to schedule events.
-  Uses TextRaction AI to intelligently identify event-like content.
-  No repeated logins required – one-time calendar access.

### Built With

- HTML / CSS / JavaScript
- Chrome Extensions API
- Google Calendar API
- TextRaction AI

### Use Case

Helps professionals and students automatically track meetings, deadlines, and events mentioned in their emails without any manual effort.
